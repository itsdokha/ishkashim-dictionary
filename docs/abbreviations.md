# Условные сокращения и иноязычные формы в словаре

В карточке каждого слова есть блок **«Этимология»** и **«Комментарий»**. Там слово сравнивается с родственными словами (когнатами) в других иранских языках, чтобы показать его происхождение. Сокращение перед формой — это **название языка-источника**, а сама форма (например, `zanāχ`, `warok`, `afau`) — слово этого языка с тем же или близким значением.

> **Как читать.** Запись «вах. `zanāχ`» в статье *alosha* «подбородок, челюсть» означает: *в ваханском языке это же понятие выражается словом* `zanāχ`. То есть иноязычная форма, как правило, значит **то же, что и заглавное ишкашимское слово** (если в кавычках «…» не указано иное значение).

## 1. Сокращения языков и диалектов

| Сокращение | Язык | Пояснение |
|---|---|---|
| забак. | зебакский | Говор, очень близкий ишкашимскому; часто приводится для прямого сравнения. |
| сангл. | сангличский | Диалект, ближайший родственник ишкашимского (ишкашимско-сангличская группа). |
| вах. | ваханский | Памирский язык; одна из ближайших параллелей ишкашимскому. |
| язгул. | язгулямский | Памирский язык верховьев Язгулема. |
| шугн. | шугнанский | Крупнейший памирский язык (шугнано-рушанская группа). |
| мундж. | мунджанский | Памирский язык; вместе с юдгским образует мунджано-юдгскую подгруппу. |
| юдг. | юдгский (юдга) | Близок мунджанскому. |
| тадж. | таджикский | Близкородственный язык-сосед, основной источник современных заимствований. |
| перс. | персидский | Новоперсидский (фарси); источник книжных и культурных заимствований. |
| араб. | арабский | Культурные и религиозные заимствования, часто пришедшие через персидский. |
| араб.-перс. | арабско-персидский | Арабское по происхождению слово, усвоенное через персидский. |
| тюрк. | тюркский | Узбекский и другие тюркские; бытовые заимствования. |
| авест. | авестийский | Древний восточноиранский язык зороастрийских текстов — опора иранской этимологии. |
| др.-перс. | древнеперсидский | Древний западноиранский язык клинописных надписей Ахеменидов. |
| иран. | (пра)иранский | Реконструированный язык-предок; формы помечаются звёздочкой (*). |
| др.-инд. | древнеиндийский (санскрит) | Привлекается для древних индоиранских параллелей. |

## 2. Служебные пометы и знаки

| Помета / знак | Значение |
|---|---|
| ср. | сравни (приводится для сравнения) |
| заимств. | заимствовано (из указанного языка) |
| √ | глагольный корень |
| * | реконструированная, незасвидетельствованная форма (праязык) |
| > | переходит в, даёт (звуковое изменение) |
| < | происходит из, восходит к |
| «…» | значение слова/формы |
| Пахалина | Т. Н. Пахалина — автор основных научных работ по ишкашимскому языку; источник многих форм |
